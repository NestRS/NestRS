# nest-rs-codegen

Internal — shared syn/quote helpers every nestrs *-macros crate is built from; not a direct dependency (third-party macro crates may also use it).

Internal proc-macro / support crate for **[NestRS](https://nestrs.dev)**.
**Do not depend on this crate directly** — use [`nest-rs-macros`](https://crates.io/crates/nest-rs-macros) instead.
