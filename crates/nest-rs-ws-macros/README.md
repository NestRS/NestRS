# nest-rs-ws-macros

Internal — WebSocket gateway decorator macros (#[gateway], #[messages], #[subscribe_message]); re-exported by nest-rs-ws, not a direct dependency.

Internal proc-macro / support crate for **[NestRS](https://nestrs.dev)**.
**Do not depend on this crate directly** — use [`nest-rs-ws`](https://crates.io/crates/nest-rs-ws) instead.
