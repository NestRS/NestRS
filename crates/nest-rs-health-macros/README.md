# nest-rs-health-macros

Macro crate for the NestRS framework — the #[indicators] decorator macro; re-exported by nest-rs-health: depend on that crate, not this one.

Part of [NestRS](https://nestrs.dev) — every framework crate ships at `1.0.0` in lockstep, under a semver contract: breaking changes wait for `2.0`.
