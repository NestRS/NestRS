# nest-rs-resource

Expose a SeaORM entity to REST/OpenAPI (wire DTO) from one declaration, via the #[expose] attribute; optional GraphQL surface behind the `graphql` flag + feature.

Part of [NestRS](https://nestrs.dev) — every framework crate ships at `1.0.0` in lockstep, under a semver contract: breaking changes wait for `2.0`.

```toml
[dependencies]
nest-rs-resource = "1.0"
```
