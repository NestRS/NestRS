# nest-rs-resource

Expose a SeaORM entity to GraphQL + OpenAPI from one declaration, via the #[expose] attribute.

Part of **[NestRS](https://nestrs.dev)** — an opinionated Rust framework for backend apps.

## Install

```toml
nest-rs-resource = "0.1"
```
