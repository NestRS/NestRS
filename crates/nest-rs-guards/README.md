# nest-rs-guards

Transport-spanning guards for NestRS — one trait, three transports (HTTP, GraphQL, WS), declared once with App::builder().use_guards_global(...).

Part of [NestRS](https://nestrs.dev) — every framework crate ships at `1.0.0` in lockstep, under a semver contract: breaking changes wait for `2.0`.

```toml
[dependencies]
nest-rs-guards = "1.0"
```
