# nest-rs-schedule

Discovered cron-style scheduled jobs for nestrs: the #[scheduled] decorator plus a ScheduleModule that auto-attaches the Scheduler.

Part of **[NestRS](https://nestrs.dev)** — an opinionated Rust framework for backend apps.

## Install

```toml
nest-rs = { version = "0.1", features = ["schedule"] }
```
