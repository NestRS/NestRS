# nest-rs-core

Core building blocks (DI container, module trait, lifecycle, access graph) for the nestrs framework.

Part of **[NestRS](https://nestrs.dev)** — an opinionated Rust framework for backend apps.

## Install

```toml
# Usually via the umbrella crate:
nest-rs = { version = "0.1", features = ["http"] }
```
