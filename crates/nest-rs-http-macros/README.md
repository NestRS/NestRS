# nest-rs-http-macros

Internal — HTTP decorator macros (#[controller], #[routes], verb attributes, #[interceptor]); re-exported by nest-rs-http, not a direct dependency.

Internal proc-macro / support crate for **[NestRS](https://nestrs.dev)**.
**Do not depend on this crate directly** — use [`nest-rs-http`](https://crates.io/crates/nest-rs-http) instead.
