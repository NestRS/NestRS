# nest-rs-http

HTTP transport for NestRS: HttpTransport, Controller trait and HttpControllerMeta, built on top of poem.

Part of [NestRS](https://nestrs.dev) — every framework crate ships at `1.0.0` in lockstep, under a semver contract: breaking changes wait for `2.0`.

```toml
[dependencies]
nest-rs-http = "1.0"
```
