# nest-rs-mcp

Model Context Protocol server support for nestrs — wraps the rmcp SDK behind the nestrs DI container and exposes a poem endpoint.

Part of **[NestRS](https://nestrs.dev)** — an opinionated Rust framework for backend apps.

## Install

```toml
nest-rs = { version = "0.1", features = ["mcp"] }
```
