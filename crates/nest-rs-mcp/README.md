# nest-rs-mcp

Model Context Protocol server support for NestRS — wraps the rmcp SDK behind the DI container and exposes a poem endpoint.

Part of [NestRS](https://nestrs.dev) — every framework crate ships at `1.0.0` in lockstep, under a semver contract: breaking changes wait for `2.0`.

```toml
[dependencies]
nest-rs-mcp = "1.0"
```
