# nest-rs-macros

Internal — surface-agnostic nestrs decorators (#[injectable], #[module], #[hooks]); re-exported by nest-rs-core, not a direct dependency.

Internal proc-macro / support crate for **[NestRS](https://nestrs.dev)**.
**Do not depend on this crate directly** — use [`nest-rs-core`](https://crates.io/crates/nest-rs-core) instead.
