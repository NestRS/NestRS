# nest-rs-worker

Worker-execution primitives for NestRS — the ambient-context seam every transport that runs jobs off the request path (schedulers, queue workers, stream consumers) plugs into.

Part of [NestRS](https://nestrs.dev) — every framework crate ships at `1.0.0` in lockstep, under a semver contract: breaking changes wait for `2.0`.

```toml
[dependencies]
nest-rs-worker = "1.0"
```
