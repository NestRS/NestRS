# nest-rs-graphql-macros

Internal — GraphQL decorator macros (#[graphql], #[resolver]); re-exported by nest-rs-graphql, not a direct dependency.

Internal proc-macro / support crate for **[NestRS](https://nestrs.dev)**.
**Do not depend on this crate directly** — use [`nest-rs-graphql`](https://crates.io/crates/nest-rs-graphql) instead.
