# nest-rs-openapi

OpenAPI 3.0 document generation and bundled Swagger UI for NestRS HTTP controllers (the REST analog of nest-rs-graphql's GraphqlModule).

Part of [NestRS](https://nestrs.dev) — every framework crate ships at `1.0.0` in lockstep, under a semver contract: breaking changes wait for `2.0`.

```toml
[dependencies]
nest-rs-openapi = "1.0"
```
