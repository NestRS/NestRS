# nest-rs-graphql

GraphQL resolver discovery + schema composition for NestRS apps, on top of async-graphql.

Part of [NestRS](https://nestrs.dev) — every framework crate ships at `1.0.0` in lockstep, under a semver contract: breaking changes wait for `2.0`.

```toml
[dependencies]
nest-rs-graphql = "1.0"
```
