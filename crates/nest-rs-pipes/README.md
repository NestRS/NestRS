# nest-rs-pipes

Transport-agnostic validation & transformation pipes for nestrs. A surface (HTTP, …) applies them at its request boundary.

Part of **[NestRS](https://nestrs.dev)** — an opinionated Rust framework for backend apps.

## Install

```toml
nest-rs = { version = "0.1", features = ["http"] }  # re-exported by nest-rs-http
```
